## Requirements
* nodejs
* npm
* mongodb-server

### Clone project:
``git clone https://github.com/chestnyh/diot-test.git``

### Change directory:
``cd diot-test``

### Init project:
``npm i``

### Build project:
``npm run build``

### Configure project:
1)``cp .env.example .env``

2)Open .env file and set current variables. IT IS NECESSARY TO CONNECT TO DATABASE

### Start project
``npm run start``
